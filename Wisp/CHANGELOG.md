# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]


### Added
- Initial support for four contexts: narrative, dashboard, form, minimal
- Auto-fetcher CLI tool for processing any URL
- Python scanner engine with BeautifulSoup
- JavaScript runtime for browser
- Minification support (jsmin, rcssmin)

## [0.1.0] - 2024-01-XX

### Added
- Core Wisp scanner with density analysis
- Context detection algorithm
- Pattern recognition (prose, structured, technical, navigational)
- CSS custom property generation
- Base styles for all contexts
- CLI bake tool for static CSS generation
- Auto-fetcher for live URL processing
- Test suite with fixtures
- Build automation script
- GitHub and GitLab repository setup
- Wikipedia demo example

### Features
- Zero-dependency runtime (2KB JS)
- Zero-config operation
- Semantic HTML-first approach
- Automatic accessibility enhancements
- Dark mode support (prefers-color-scheme)
- Reduced motion support
- Print optimization
